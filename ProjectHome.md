## AcChat is a twisted powered chat server ##
  * aimed to serve a PAD like Flex/Flash player (e.g. [PADplayer](http://code.google.com/p/padplayer)) with flying comments effects
  * using a self-designed json-based protocol(see API)
  * developed by observer(jingchaohu@gmail.com)

The name "AcChat" comes from "Anime Comic Chating Server"

## AcChat and Avfun ##
  * This project is a sub project of [avfun](http://code.google.com/p/avfun) project
